# ☸️ Kubernetes for DevOps

Welcome to my **Kubernetes for DevOps** learning repository.

This repository documents my hands-on journey to mastering **Kubernetes**, the industry-standard container orchestration platform. Instead of only learning concepts, I deploy applications, manage clusters, configure networking and storage, and build production-ready Kubernetes environments for modern DevOps workflows.

---

# 🎯 Goal

* Understand Kubernetes fundamentals
* Learn Kubernetes architecture
* Deploy and manage containerized applications
* Master Pods, Deployments, Services, and Ingress
* Learn Kubernetes networking and storage
* Build real-world Kubernetes projects
* Prepare for production Kubernetes environments

---

* **Operating System:** Windows 11
